TUTORIAL GIT

¿QUÉ ES GIT BASH?

Git Bash es una aplicación para entornos
Windows que proporciona una capa de
emulación para un entorno de comandos
de Git. Esta herramienta combina el
poder de Git con una interfaz de línea de
comandos bash, que es común en los sistemas
operativos Unix/Linux.

¿PARA QUÉ SIRVE GIT BASH?

Git Bash sirve para interactuar con Git
usando comandos de texto. Al ser una
interfaz de línea de comandos, permite
a los usuarios realizar diversas tareas.

Este conjunto de comandos cubre la configuración inicial de Git, la creación y manejo de un repositorio, la adición y commit de archivos, la fusión de cambios desde el remoto y la resolución de errores comunes.
